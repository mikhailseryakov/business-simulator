# Business Simulator — Тренажёр управленческих решений

Интерактивная веб-платформа для тренировки управленческих решений через симуляции на Python/Flask.

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить приложение
python app.py

# 3. Открыть в браузере
# http://localhost:5000
```

## Структура проекта

```
business_simulator/
├── app.py                      # Flask-приложение (бэкенд + логика)
├── requirements.txt
├── README.md
├── templates/
│   ├── index.html              # Главная страница
│   ├── scenarios.html          # Выбор сценария
│   ├── simulation.html         # Интерфейс симуляции
│   └── results.html            # Результаты и аналитика
├── static/
│   ├── css/style.css           # Стили (dark corporate theme)
│   └── images/                 # Сгенерированные графики
├── scenarios/
│   ├── launch_product.json     # Запуск продукта
│   ├── crisis_management.json  # Антикризисное управление
│   └── new_market.json         # Выход на новый рынок
└── data/
    └── user_progress.json      # Прогресс пользователей
```

## Сценарии

| Сценарий | Сложность | Бюджет | Решений |
|---|---|---|---|
| Запуск нового продукта | Средний | 1 000 000 ₽ | 4 |
| Антикризисное управление | Сложный | 2 000 000 ₽ | 4 |
| Выход на новый рынок | Эксперт | 3 000 000 ₽ | 4 |

## Формула оценки

```
Итоговый балл = Бюджет×0.3 + Доля рынка×0.5 + Репутация×0.2
```

| Диапазон | Оценка |
|---|---|
| > 80 | Отлично 🏆 |
| 60–80 | Хорошо 👍 |
| 40–60 | Удовлетворительно 📊 |
| < 40 | Неудовлетворительно ⚠️ |

## Деплой на VPS

```bash
# Установить gunicorn
pip install gunicorn

# Запустить в продакшн-режиме
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

## Зависимости

- Python 3.8+
- Flask >= 2.3.0
- Matplotlib >= 3.7.0
- Только стандартные библиотеки Python
